# 隐私

Codex Monitor 是一个本地工具。

## 它会读取什么

- `~/.codex/sessions` 下的 Codex Desktop 会话日志。
- 你选择作为提示音的本地音频文件。
- 保存在 `UserDefaults` 中的用户偏好设置。

## 它不会做什么

- 不上传 Codex 日志。
- 不调用云端账单或用量 API。
- 不发送遥测数据。
- 不修改 Codex Desktop 设置。

## 重要说明

Codex 会话日志可能包含 prompt、文件路径、命令输出和其他本地隐私上下文。本应用只在本机解析这些文件，用来提取事件元数据、用量计数、完成与失败信号。

提交公开 issue 时，不要直接附上未脱敏的原始会话日志。
